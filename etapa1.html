Como funciona o computador

O processo de inicialização de um computador segue uma ordem fixa e independente do sistema operacional
em uso. Quando pressionamos o botão de energia, a corrente elétrica é ativada, alimentando os componentes da máquina. Em seguida, o BIOS (ou UEFI) entra em ação, realizando o POST (Power On Self Test) para verificar o funcionamento dos componentes.

Após a verificação bem-sucedida, o BIOS/UEFI procura um dispositivo para iniciar o sistema operacional. Isso pode ser um disco rígido,
SSD ou até mesmo um Live-DVD com um sistema Linux para instalação, dependendo da configuração do BIOS/UEFI. O bootloader, localizado no MBR ou na partição EFI, é então carregado, e ele é responsável por iniciar o sistema operacional ou oferecer opções de escolha, como no caso de sistemas com múltiplas instalações.

O bootloader desempenha um papel crucial no processo de inicialização, pois é responsável por carregar o sistema operacional específico instalado
no computador. Isso significa que, se você tiver vários sistemas operacionais em sua máquina, o bootloader permitirá que você escolha quais desejam iniciar. Por exemplo, os usuários de sistemas Linux frequentemente utilizam o GRUB para selecionar entre diferentes distribuições ou versões do sistema. Esse passo é fundamental para a flexibilidade e funcionalidade de sistemas com configurações múltiplas.

Uma vez que o sistema operacional é escolhido e carregado, uma interface de login é exibida, permitindo que o usuário acesse a máquina. A partir deste ponto, o sistema operacional assume o controle total e disponibiliza todos os recursos e aplicativos. É importante notar que, embora os detalhes do processo de inicialização possam variar entre os sistemas operacionais e as configurações
específicas da máquina, a sequência geral descrita aqui permanece consistente em computadores modernos.
